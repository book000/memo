# VRChat におけるゲーム内設定とレジストリ値

VRChat の設定はほぼすべてがレジストリに登録されている。この記事ではゲーム内での設定項目と、その設定がどのレジストリ項目に反映されるかを調査してまとめたもの。  
なお、一部の設定値は `reg query` コマンドでは正しい値が出力されないため注意（`reg export` を用いて調査した）

!!! info "執筆途中"
    この記事は執筆途中です。情報共有をお待ちしています。
